# Interactive Physics Simulation 🕸️

A dynamic, interactive 2D physics engine built using **Vanilla JavaScript** and **HTML5 Canvas**. This project features a Verlet physics simulation where customizable wires/strings physically react to user cursor movements, gravity, and friction in real-time.

### 🚀 Live Demo
**[Experience the Simulation Here](https://pankajtiwari-art.github.io/Interactive-Physics/)**

---

## ✨ Features

* **Real-Time Physics Engine:** Smooth Verlet integration handling momentum, gravity, and drag.
* **Fully Interactive:** Wires dynamically stretch, bounce, and follow the user's cursor or touch input.
* **Custom Entity & Background:** Upload your own background cover images and cursor entity images directly from the UI.
* **Advanced Configuration Panel:**
  * Adjust **Gravity**, **Air Friction**, and **Segment Length** on the fly.
  * Control the **Wire Count** and custom **Colors** (Background & Wires).
  * Scale the cursor object and adjust its anchor connection points.
* **Responsive Design:** Works seamlessly on both desktop browsers and mobile touch screens.
* **Zero Dependencies:** Built entirely with pure HTML, CSS, and vanilla JavaScript. No external libraries.

---

## 🛠️ Technologies Used

* **HTML5** (Canvas API)
* **CSS3** (Custom UI Controls & Styling)
* **JavaScript (ES6+)** (Object-Oriented Physics Logic, RequestAnimationFrame loop)

## 💻 How to Run Locally

Since this project has zero dependencies, running it locally is incredibly simple:

1. **Clone the repository:**
   ```bash
   git clone [https://github.com/PankajTiwari-Art/Interactive-Physics.git](https://github.com/PankajTiwari-Art/Interactive-Physics.git)
   ```
2. **Navigate to the directory:**
   ```bash
   cd Interactive-Physics
   ```
3. **Open `index.html`:**
   Simply double-click `index.html` to open it in your default web browser.

---

## 🎮 How to Use

1. Move your mouse (or drag your finger on mobile) across the screen to interact with the physics object.
2. Click the **"Settings"** button in the bottom-right corner to open the configuration panel.
3. Use the **Upload** buttons to change the background or the object being dragged.
4. Play with the **Physics sliders** (Gravity, Friction, Wire Count) to see how the engine reacts in real-time!
